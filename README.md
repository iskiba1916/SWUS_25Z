# SWUS_25Z
Repozytorium zawierające dokumentację projektu „Zarządzanie politykami sieciowymi w systemie Kubernetes” zrealizowanego na Wydziale Elektroniki i Technik Informacyjnych PW. Skupia się na implementacji izolacji sieciowej w architekturze trójwarstwowej przy użyciu CNI Cilium oraz weryfikacji bezpieczeństwa.
